Web Design Explained

The relationship between core web technologies is comparable to the components of a sound, livable, working home

HTML is like a home's structure

Web page developers depend on standard, tried-and-true parts for the layout of web pages, similar to the way builders depend on parts and materials to construct houses.

HyperText Markup Language's "block" level components and semantic elements can be viewed as similar to the beams, bars, walls and floors that make a residence physically stable and structurally sound.

CSS is like a home's decor

Once a web page's structure is intact, web designers can focus on interior decoration, so to speak.

Cascading Stylesheets allow designers to establish efficient rules and preferences that are similar to the design choices that give a home its look, feel, and appeal. Good design plays a big role in good user experience (UX) .

JavaScript is like a home's appliances

For anyone to live and operate within a home, numerous things in the home have to work when used.

JavaScript's "events" are similar to what happens when people flip a switch, i.e. does the light go on or off as expected? Does the oven work? Does the air conditioner work? How about the sink? Web developers use JavaScript to determine how a website will function and to guide user interaction (UI).

Content is like a home's family

Just as the purpose of a home is to house real people, the purpose of a website is to house relevant content.

Information, media, data and meaning are the real reason why users visit a website, plus return to it or share it with others. The content that lives on a site is arguably the most important element of all.
